Principais formatações

Negrito/Destaque

Para texto em Negrito, use tag "b"

Para texto com Destaque, use a tag "strong"

Itálico/Destaque

Para texto em italico, use a tag "i"

Para texto com enfase, use a tag "Em"

Texto marcado

Para texto com algo marcado, use a tag "mark"

Texto grande e pequeno

Para criar um texto grande, use a tag

Para criar um texto pequenno, use a tag "small"

Texto deletado

podemos marcar um texto como excluido para indicar que ele deve ser lido mas não considerado, se usa a tag "del"

Texto inserido

podemos marcar um texto como inserido para dar enfase e indicar que ele foi adicionado depois, se usa a tag "ins"

texto sobrescrito e subscrito

sobreescrito é para colocar coisas como x10 = 200, se usa a tag "sup"

subscrito é para colocar coisas como H2O, se usa a tag "sub"
